Listas Ordenadas (ORDER LIST "OL")

Listas ordenadas são a tag "OL" order list

  1. Acordar
  2. Tomar café
  3. Ligar para o João

De acordo com o HTML5, o "/LI" (fechamento da tag "LI") se tornou opcional.
Também temos a opção type na tag "OL", para organizar a lista de acordo com o seu gosto, por exemplo:

Se utilizar o "a ou A" no type, a lista será organizada em (a,b,c,d,e).

Se colocarmos o "I ou i", a lista ficara em romanos (I, II, III, IV, V, VI)


Lista não ordenadas (UNORDERED LIST "UL")

Uma lista não ordenada, o próprio nome já diz, não tem uma sequência.

  • Acordar
  • Tomar um banho
  • Ir treinar
  • Limpar a casa
  • Estudar

Também temos alguns estilos para esse tipo de lista, podemos alterar o formado acrescentando um type na tag ul" e adicionando os valores circle, disc,square.
