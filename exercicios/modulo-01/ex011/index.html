Inserindo vídeos hospedados localmente

Este vídeo está hospedado no meu príprio servidor. Para isso, use a tag "video" e assim como a tag "audio" temos algumas propriedades, como:

  • Autoplay
  • Controls
  • Width
  • Poster - indica a capa do vídeo
  • Loop
