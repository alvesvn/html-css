Usando links externos

Link que vai apontar para outro site.

Você pode acessar o meu repositório público no GitHub.

Sempre é recomendado usar o "target:_blank" e o "rel="external".


Usando links internos

Link que vai fazer uma relação com outra página que esta dentro do nosso servidor.

Está é a primeira página do site, se você quiser pode visitar a minha segunda página.

Lembre-se que temos o o rel e seus atributos:

  • External
  • Prev
  • Next
  • Nofollow
Propriedades target
  • _blank
  • self

Para caminhar entre as páginas utilize:

  • tag "a" no href "./" ou ctrl+espaço
  • Para voltar algumas pastas coloque "../" dois pontos

Você também pode acessar a nossa página de notícias.
